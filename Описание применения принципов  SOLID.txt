Принцип единственной обязанности
- класс Setting: настройки Interval и Attempt
- класс Interval: Min, Max
- базовый класс GueesReaderBase: единственный метод для ввода
- класс наследник GueesReaderInterval: для ввода Interval
- класс наследник GueesReaderNumber: для ввода number в пределах Interval
- класс наследник GueesReader: для хранения Setting, Number
- класс GueesValidator: единственный метод для проверки угадываемого числа
- класс GueesGenerate: единственный метод для генерации случайного числа в заданных пределах
- класс GueesReaderAttempt наследует GueesReaderBase, IGueesReader
    имеет метод Get(Setting setting) - реализация для получения количества попыток
	
Принцип открытости/закрытости
- класс GueesReader наследует GueesReaderBase
    имеет метод Get(IGueesReader gueesReader) - реализация для получения числа
- вызов Reader.Get(new GueesReaderNumber()) - вызов IGueesReader
- вызов Reader.Get(new GueesReaderAttempt()) - тоже
	
Принцип подстановки Лисков
- класс GueesReaderAttempt реализует IGueesReader метод Get
используется в классе GameStore как var _attempt = Reader.Get(IGueesReader) - new GueesReaderAttempt()
- класс GueesReaderNumber реализует IGueesReader метод Get
используется в классе GameStore как var _number = Reader.Get(IGueesReader) - new GueesReaderNumber()
